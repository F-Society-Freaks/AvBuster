Stress test your Anti Malware / End Point Protection solutions.

AVBUSTER helps you to assess your anti malware solutions. It is a framework packed with options to generate different types of executable files that can be used to test any anti malware / EndPoint Security solutions on both Windows and Linux platforms. Current version supports to create following types of binaries

  • PowerShell Custom Code Injection
  • PowerShell TCP Reverse Shell
  • MSBuild TCP Reverse Shell
  • Installutil TCP Reverse Shell
  • MS XML Xslt TCP Reverse Shell
  • Forfiles TCP Reverse Shell
  • Normal TCP Reverse Shell
  • Hybrid Encrypted TCP Reverse Shell
  • HTTP Reverse Shell
  • HTTP Meterpreter Reverse Shell
  • HTTPS Pinned Cert Reverse Shell
  • HTTPS Meterpreter Reverse Shell
  • User Defined Shell Code Injection [Encrypted]
