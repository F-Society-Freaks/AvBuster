The Anti Malware Assessment Framework

Knocker is an anti malware assessment framework packed with several options that helps the user to create different types of executable files that can be used to test on both Windows and Linux platforms. Current version supports three PowerShell options to create exe files to assess Windows based anti malwares and go lang options to create both Windows and Linux executables to assess anti malwares. It is open source.
